constructor

  • new Color(color?): Color

  • Parameters

    • Optionalcolor: PaletteColor | Color | Color

      A p5.js Color, Color, or PaletteColor object.

      If given a p5.js Color, the color's RGBA components will become the values of red, green, blue, and alpha.

      If given a Color, the given color's red, green, blue, alpha, and name properties will become the new values of this color's respective properties.

      If given a PaletteColor, the color's RGB and NAME values will be used to build the color.

    Returns Color
